# Smart Intruder Detection and Alert System (S.I.D.A.S)
This project works as a smart home security system. It uses sensors and state based system logic to detect and determine it next action based on a set of predefined rules in each state. Basically its a security system that sends out alerts through multiple methods if it detects movement and/or light changes.
I made this project, well... because i wanted to try and make something that could be useful to not just me, and be able to be applied i  different circumstances and environments.
